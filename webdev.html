What is Web-Development?


Web development is the work involved in developing a Web site for the Internet (World Wide Web) or an intranet (a private network). Web development can range from developing a simple single static page of plain text to complex web applications, electronic businesses, and social network services. A more comprehensive list of tasks to which Web development commonly refers, may include Web engineering, Web design, Web content development, client liaison, client-side/server-side scripting, Web server and network security configuration, and e-commerce development.

Wikipedia




Which Tools are required for Web-Development?


There are many languages which are used for website development. They include HTML, CSS, JavaScript, php, Python etc. There are also popular frameworks and libraries which help the developers to develop websites quickly and efficiently. They include Bootstrap, React JS, Angular, Node JS, Django etc.

Bootstrap React JS Angular Django Node.js
Bootstrap is a free and open-source CSS framework directed at responsive, mobile-first front-end web development. It contains CSS- and (optionally) JavaScript-based design templates for typography, forms, buttons, navigation, and other interface components.

Wikipedia
React is an open-source, front end, JavaScript library for building user interfaces or UI components. It is maintained by Facebook and a community of individual developers and companies. React can be used as a base in the development of single-page or mobile applications. However, React is only concerned with state management and rendering that state to the DOM, so creating React applications usually requires the use of additional libraries for routing, as well as certain client-side functionality.

Wikipedia
Angular is a TypeScript-based open-source web application framework led by the Angular Team at Google and by a community of individuals and corporations. Angular is a complete rewrite from the same team that built AngularJS.

Wikipedia
Django is a Python-based free and open-source web framework that follows the model-template-views (MTV) architectural pattern. It is maintained by the Django Software Foundation (DSF), an American independent organization established as a non-profit.

Wikipedia
Node.js is an open-source, cross-platform, back-end JavaScript runtime environment that runs on the V8 engine and executes JavaScript code outside a web browser. FAQs


  • What is an IDE?: IDE stands for Integrated Development Environment. It is a software application which provides the user with a text editor, a language translator and debugging tools.

  • What is an interpreter?: It is a language translator which translates the code of an interpreted languages into bytecode line by line.

  • Is JavaScript an interpreted language?: Yes JavaScript is an interpreted language.

  • Is form validation a part of backend? No! the basic form validation done with JavaScript code lies under frontend development. Backend development is about servers and databases only.

  • What is the use of JavaScript in frontend development?: Drobdown menus, working buttons, form validation, logic etc, all these things are done by using JavaScript.

  • How long does one web development project take to complete from start to finish?: It takes 6 to 12 weeeks in making a complete website with custom code.

  • What is the use of Sass language?: Sass is a meta-language on top of CSS that's used to describe the style of a document cleanly and structurally, with more power than flat CSS allows.
