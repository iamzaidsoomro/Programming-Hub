Q: Which languages are used in videogame development?

ANS: C++, JavaScript, Python are the most popular languages used in game development.



Q: What is a game engine?

ANS: It is an environment provided to the game developer for the development and testing of a game.



Q: How can we develop games in python?

ANS: There's a library in python, which is called pygame. This library is used to create videogames.

Q: Which language is used in Unreal Engine?

ANS: C++.



Q: How are 3d game models made?

ANS: There are soem high-end animation software like Autodesk, these type of software are used to create 3d models in a game.

Q: Who are game designers?

ANS: The game designers work with the creative aspects of a videogame for example story, dialogues etc.



Q: What is the difference between a game developer and a game designer?

ANS: A videogame designer is the one who deals with the creative aspects of the game while a developer actually implement the ideas through coding.

Q: Who is a game tester?

ANS: A games tester is the guy who works for video game production companies to thoroughly test video games before the final version is released to the public.
